# Library Management System

## Project Overview

Library Management System is a web-based application developed using PHP, MySQL, HTML, CSS, and JavaScript. The system is designed to automate and manage daily library operations such as book management, member management, issue and return of books, fine management, notifications, suppliers, purchases, payments, and report generation.

## Installation

### Step 1
Install XAMPP.

### Step 2
Copy the project folder into:

```txt
xampp/htdocs/
```

### Step 3
Start:

- Apache
- MySQL

from XAMPP Control Panel.

### Step 4
Open phpMyAdmin and create a database:

```sql
library_management
```

### Step 5
Import the SQL file into the database.

### Step 6
Open your browser and run:

```txt
http://localhost/library_management
```
